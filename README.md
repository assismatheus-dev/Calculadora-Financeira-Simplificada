# Calculadora Financeira Simplificada

Sistema desenvolvido em Java com foco na aplicação de juros sobre um montante inicial, permitindo o cálculo do valor final e a geração de parcelas.

## Objetivo

Simular operações financeiras básicas utilizando:

- Montante inicial
- Taxa de juros
- Período (tempo)
- Geração de parcelas

O projeto foi criado com fins educacionais para reforçar conceitos de lógica de programação e organização orientada a objetos.

## Conceitos aplicados

- Estruturas condicionais
- Laços de repetição
- Métodos
- Organização em classes
- Cálculo de juros simples e/ou compostos

## Fórmulas Utilizadas

Juros Simples:

M = C × (1 + i × t)

Juros Compostos:

M = C × (1 + i)^t

Onde:

M = Montante final  
C = Capital inicial  
i = Taxa de juros  
t = Tempo  

## Como executar

1. Clone o repositório:

git clone https://github.com/assismatheus-dev/Calculadora-Financeira-Simplificada.git

2. Compile o projeto:

javac Main.java

3. Execute:

java Main
